Разработать клиента с возможностью подключения к серверу и обмена сообщениями

Задание на 3
Разработать сервер с возможностью одновременного подключения нескольких клиентов.
Вести общий чат.
Хранить на сервере историю чата, а также всех пользователей с логинами и паролями, историю передавать клиенту при подключении
Разработать клиента с возможностью подключения к серверу и обмена сообщениями

Задание на 4
Все то же, что на 3.
Дополнительно для сервера:
 - возможность организовать чат из двух клиентов
 - посылать уведомление клиенту, когда в сети появился его собеседник
 - хранить историю каждого чата отдельно
Дополнительно для клиента:
 - возможность организации закрытого чата с другим клиентом
 
Задание на 5
Все то же, что на 4.
Дополнительно для сервера:
 - хранить список друзей, с кем можно организовать чат
 - организовывать чаты с любым числом друзей
Дополнительно для клиента:
 - возможность управлять друзьями
 - возможность организации чата с несколькими друзьями

2 вариант
Разработка игры

Задание на 3
Разработать сервер с возможностью одновременного подключения двух клиентов.
На сервере хранить историю игры, передавать от клиента клиенту команды
Разработать клиента с отображением интерфейса  игры и подключением к серверу для обмена командами

Задание на 4
Все то же, что на 3.
Дополнительно для сервера:
 - клиентов может быть много, но играют они парами (один против одного)
 - посылать уведомление клиенту, когда в сети появился его соперник
 - хранить историю каждой игры отдельно
 - для клиентов хранить логины-пароли
Дополнительно для клиента:
 - возможность выбора соперника
 
Задание на 5
Все то же, что на 4.
Дополнительно для сервера:
 - игроки могут выбрать соперника только из какого-то списка (друзья, рейтинг - любой критерий) 
 - если позволяет игра, то играть можно командами
 - рейтинг игроков
